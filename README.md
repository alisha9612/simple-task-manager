# Simple Task Manager

A simple and user-friendly Task Manager web application built using HTML, CSS, and JavaScript. This application helps users manage daily tasks by adding, completing, and deleting tasks.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Add tasks using the Enter key
* Clean and responsive user interface
* Visual strikethrough effect for completed tasks

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation & Event Handling)

## Functionality

### Add Task

* Enter a task in the input field.
* Click the **Add** button or press **Enter**.
* The task will be added to the task list.

### Complete Task

* Click the **Complete** button.
* The task will be marked as completed with a strikethrough effect.

### Delete Task

* Click the **Delete** button.
* The selected task will be removed from the list.
